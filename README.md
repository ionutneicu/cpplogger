# cpplogger
A simple C++ 11 logging module

## This module is a simple modern C++logging module designed for application written in C++11 or newer

### Features

- simple to use, just one .h and one .cpp file 
- simple to understannd
- easy to extend and adapt to your requirements


### Compilation - default, to console

```bash
mkdir build
cd build
cmake <path-to-this-repo-cmake_file>
make
```

### Run Test Application

```bash
./examples/run_tests
```
